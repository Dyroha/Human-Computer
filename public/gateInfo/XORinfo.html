The XOR gate (exclusive OR gate) is similar to the OR however it does not output a black flag '🏴' if both inputs are black flags '🏴'
Instead it will output a white flag '🏳️'

So it will...
output a black flag '🏴' if either input is a black flag '🏴'
but if both inputs are black '🏴' or white '🏳️' flags it will output a white flag '🏳️'
